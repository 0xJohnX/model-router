# Model Router v1.0.0 Release

## 核心亮点

**问题**: 大多数 Agent 对所有任务都用昂贵的大模型，浪费 token。

**方案**: 智能路由——简单任务用 Haiku（便宜）、标准工作用 Sonnet（平衡）、复杂推理用 Opus（深度）。

**成果**: 成本降低 60-80%，同时保持输出质量。

## 三档分类

| 级别 | 模型 | 价格 | 适用场景 |
|------|------|------|---------|
| 🟢 **Fast** | Haiku 4.5 | $1-$5/M | 查询、格式化、翻译、简单Q&A、heartbeat、定时任务、子任务 |
| 🟡 **Balanced** | Sonnet 4.6 | $3-$15/M | 写作、分析、代码、研究、产品思考、交互对话 |
| 🔴 **Deep** | Opus 4.6 | $5-$25/M | 架构决策、安全审查、复杂调试、高风险输出 |

## 关键特性

✅ 简洁规则 — 只有 3 档，Agent 容易理解和执行  
✅ 自动升降级 — 失败时升级，完成后自动降回  
✅ Per-Agent 预设 — 设计Bot默认Fast、代码Bot默认Balanced  
✅ 子任务优化 — 大多数成本节省来自派发任务时用Haiku  
✅ 静默模式 — 快速批量决策时不用每次都确认  
✅ 成本透明 — Agent 知道价格差异（1x/3x/5x），辅助决策  
✅ 人在环路 — 建议不强制，用户始终掌控
